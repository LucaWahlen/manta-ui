# Manta UI

A modern web component library built with Stencil.

## Installation

```bash
npm install manta-ui
```

## Quick Start

```js
import 'manta-ui/css';
import 'manta-ui';
```

```html
<manta-button>Click Me</manta-button>
```

## Development

```bash
npm install
npm start
npm run storybook
```
